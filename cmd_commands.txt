=========================Команды на линуксе=========
pwd- показывает путь к текущей папке
ls- показывает содержимое папки 
ls - a- показывает скрытые файлы
ls- l - подробное описание файлов и директорий

nano- тесктовый редактор
rm- удаляет файлы
rm -r - рекурсивное удаление файлов
rm -rf- принудительное удаление всех файлов
mkdir- создание новой директорий (mkdir project)
touch- создание нового файла (в конце указывем расширение файла- touch file.txt)
cd- сменит папку на домашнюю (папку пользлвателя)
cd / - переместит в корневую папку
cd .. - возвращает на шаг назад
cd ... - возвращает на два  шага назад
cd ../.. - возвращает на два  шага назад
cat - выведет все содержимое файла в терминал 
cp - копирование файла (cp что копируем, куда копируем cp file.txt /os/ubuntu)
mv - перемещает/переименовать файл в указанную директорию (mv file.txt / os/windows)
echo - вывод на экран терминала сообщение
echo сообщение > файл - запись сообщения в наш файл
man - manual или инструкция по командам (man echo)
rmdir - удаление пустой папки 

Nano команды:
ctrl + O - мы сохраняем все что написали (enter)
ctrl + G - это мануал для нано
ctrl + X - выход из редактора 

history- это команда которая отображает список последних выполненных команд в истории сессии терминала

С помощью знака > мы можем записать всю историю в файл 

grep - это утилита в командной строке, предназначенная для текста в файлах 
grep 'example' /path/to/example.txt

history| grep название_команды

clear - очищает терминал от предыдущих сообщений

sudo shutdown -h now- выключатель компьютер немедленно
sudo reboot - перезагрузить компьютер
top - терминальный диспетчер задач
find- для поиска файлов по имени или расширению (find/path/to/search - name 'file.txt' - найти файл по имени)
find/path/to/search - type f-name '*txt' - найти все файлы с расширением .txt
