==============Пакетный Менеджер================

Пакетный менеджер- это инструмент, который упрощает установку, обновление, удаление и управление программами пакетами
на Операционной системе. 

Apt- (Advanced Package tool) - Пакетный менеджер в Ubuntu 

brew- пакетный менеджер в Macos

Пакет- это архив, который содержит исполняемый код программы

sudo apt update# обновить список пакетов
sudo apt upgrade# обновить установленные пакеты
sudo apt install название_пакета установит программу
sudo apt remove- название_пакета удалит программу

#Python

sudo apt install python3
sudo apt install python3-pip
python3 --version
